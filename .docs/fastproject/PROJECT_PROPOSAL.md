# プロジェクト提案書

## 1. プロジェクト名
* 
MNP契約情報管理表
## 2. 概要（エレベーターピッチ）
* このプロジェクトを一言で説明すると何か？
携帯電話MNP（番号ポータビリティ）契約に関する情報を一元管理できるCLIツール。
家族や複数回線の契約管理、各種情報の保存・編集・出力ができる。

## 3. 解決したい課題
* なぜこのプロジェクトが必要なのか？
* どのような問題や非効率を解決するのか？
エクセルやメモを管理するのは難しい為、GUIで必要な情報を入れて
ユーザが計算しなくても、必要な情報を一元管理できる。
